## How to installing?
### 1. Before that
This project is based on Docker.
If you don't have Docker installed, please install it first.

### 2. Installation these apps
If you have Docker installed, you can install these apps below.
You need select one of the 3 ways. 
(1.API Service, 2.Frontend, 3.Discord service)

curl https://jeshika0815.github.io/tom2005_webs/kyle_setup.sh | sh
